# chatgpt-telegram-bot

ChatGPT Telegram Bot

## Setup

1. Install the dependencies:
    ```bash
    npm install
    ```

2. Set env variables
    ```bash
    export OPENAI_API_KEY=<token>
    export OPENAI_API_ENABLED=<token>
    export TELEGRAM_BOT_TOKEN=<token>
    export SESSION_TIMEOUT_HOURS=<token>
    ```

3. Run the application:
    ```bash
    npm start
    ```
